## Q2 2026 - Foundation

- Rebrand core namespace from legacy references to @un0z/ui.
- Set Geist as global UI font and Geist Mono for code contexts.
- Apply corporate light/dark layout tokens for sidebar, content, and chat input.
- Migrate primary brand assets to local Un0z asset pipeline.

## Q3 2026 - Migration

- Finalize Un0z brand components and textual wordmark usage.
- Replace remaining legacy docs imports and examples.
- Stabilize metadata, favicon, and manifest for production domains.
- Validate accessibility and contrast for all theme states.

## Q4 2026 - Launch

- Publish rebrand release with migration notes and changelog highlights.
- Close remaining legacy naming debt in docs and demos.
- Expand design token governance and component visual regression checks.
- Start enterprise theming packs for partner products.
